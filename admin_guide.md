# Airbridge Admin Dashboard Guide

This guide explains how to use the Admin Dashboard to manage flights and view bookings.

## 1. Accessing the Dashboard

1.  Navigate to `login.html`.
2.  Login with the admin credentials:
    *   **Email**: `admin@airbridge.com`
    *   **Password**: `admin`
3.  You will be redirected to the Admin Dashboard (`admin.html`).

## 2. Managing Flights

The "Manage Flights" tab is the default view. Here you can see a list of all active flights.

### Adding a Flight
1.  Click the **+ Add Flight** button in the top right of the "Flight Schedule" card.
2.  A modal window will appear. Fill in the following details:
    *   **Airline**: The name of the airline (e.g., Airbridge Airways).
    *   **Flight Name**: The specific name/number of the service (e.g., Morning Express).
    *   **Flight Type**: The aircraft model (e.g., Boeing 737).
    *   **From**: Origin city/airport code.
    *   **To**: Destination city/airport code.
    *   **Date**: Date of the flight.
    *   **Dep Time**: Departure time.
    *   **Arr Time**: Arrival time.
    *   **Price**: Ticket price in USD.
3.  Click **Add Flight**.
4.  The new flight will immediately appear in the list.

### Removing a Flight
1.  Locate the flight you wish to remove in the list.
2.  Click the **Trash Icon** button on the right side of the flight row.
3.  Confirm the action in the popup dialog.
4.  The flight will be permanently removed.

## 3. Viewing Bookings
1.  Click the **All Bookings** tab at the top.
2.  You will see a list of all passenger bookings, including:
    *   Booking ID
    *   Passenger Name/Email
    *   Flight Details
    *   Status
